ngHide, ngShow, ngIf

built in Angular directives

  • Evaluates a boolean expression to determine the visibility of an element
  • ngHide and ngShow use display: none css to hide
  • ngIf removes the element from the DOM entirely
    • More appropriate when a change in state is unlikely
    • (manipulating the DOM is more costly than changing CSS properties)
