Directives

  • Directives are markup on the DOM that links parts of the DOM with some javascript
  • These can be used to attach behavior to the DOM, or to transform a DOM element and its children
  • Directives appear as custom html elements or attributes, or as css classes
  • Angular's html compiler identifies these, and links the DOM components to the javascript scope
  • Angular comes with a set of built-in directives
  • You can create your own directives
