$http

a built in Angular service

  • This service provides an easy way to make asynchronous http requests (AJAX)
  • Supports any http method (GET, POST, PUT, etc)
  • Cross site scripting prevents using these to access other domains, but also supports JSONP
  • Automatically handles sending and receiving json data, just pass in the data object
  • Shortcut methods for the common http methods
