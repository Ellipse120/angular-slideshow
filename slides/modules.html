Modules

  • Modules provide a way to package up a bundle of code to be used by other applications
  • A module is a named collection of services, directives, filters, and/or configuration information.
  • Modules are declared using angular.module()
  • A module provides a list of dependent modules as part of its declaration
  • Angular's dependency injection framework resolves the dependencies
