$location

a built in Angular service

  • This service binds to the browser's URL (based on window.location), and provides an api to work with
  • Can be used to watch and observe the URL, as well as change it
    • It does not trigger a page reload when the URL is changed.
  • Provides convenience methods to parse and work with the various parts of the url (protocol, host, port, path, search, hash)
