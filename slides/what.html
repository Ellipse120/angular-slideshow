What is it?

  • AngularJS is a full-featured, client side web framework
  • It extends the vocabulary of HTML, making it much more powerful
  • Sets up data binding between HTML and javascript
    • Allows for a separation of concerns
    • The result is very expressive and readable
    • Very quick to develop
  • A dependency injection framework
    • Allows for code to be packaged into modules
    • Makes it easy to extend and modify Angular features
    • Makes it easy to share and reuse code (ngmodules.org)
  • Much, much more...
